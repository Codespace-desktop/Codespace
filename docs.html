How to Organize & Manage Multiple Coding Projects

Codespace is a free Windows desktop app that helps you organize project files and folders, manage multiple coding projects simultaneously — in Python, Java, JavaScript, C++, C, HTML or any other language — and automate repetitive tasks on Windows, all from one control panel. This guide covers every feature with step-by-step instructions and CLI commands.

Getting Started

Setting up the CLI Backend

Codespace ships with a built-in CLI that lets you manage multiple coding projects directly from your terminal — no matter which directory you're in. Setting it up takes two steps.

Add Codespace to your System PATH

  1. 1Copy the full path of your Codespace installation directory.
  2. 2Open System Environment Variables on Windows and add the path to the Path variable.
  3. 3Open a new terminal window. You can now run codespace commands from any directory.
ℹ️ Every successfully executed CLI command outputs "success." at the end of its response when there are no errors.
Projects

Organize Project Files and Folders

Managing a proper software project folder structure is the foundation of any good workflow. Codespace stores your project metadata so you can manage multiple coding projects from one place — without hunting through Windows Explorer every time.

Create a Project GUI CLI

When you create a project, Codespace records its root directory (all files and folders used in the project) and its source directory (src — only your project files, and where imported projects land). This gives Codespace a clean software project directory structure to work from.

  1. 1Open Codespace.
  2. 2Click the Add button.
  3. 3Enter a Project name.
  4. 4Choose the Root directory — the folder that contains every file and folder used in the project.
  5. 5Choose the Source directory (src) — the folder containing only your project's code files, and where imported projects will be copied.
  6. 6Enter the Command Prompt command to run the project, including a cd command to change into the root or source directory first.
  7. 7Click OK to add the project.
CLI — Command Prompt 
C:\> codespace create <name> <rootDir> <srcDir> <runCommand>

# Example — Python project
C:\> codespace create MyPythonApp "C:\Projects\MyPythonApp" "C:\Projects\MyPythonApp\src" "cd C:\Projects\MyPythonApp && python main.py"

Works for any language — Python, Java, JavaScript, C++, C, HTML, and more.

Projects

Import a Project

When handling multiple projects simultaneously, you often need to copy one project's files into another. Codespace's import feature copies all files from one project's source folder into another project's src directory.

Import via GUI GUI CLI

  1. 1Open Codespace.
  2. 2Click on the destination project (the one you want to import files into).
  3. 3The project dashboard will open.
  4. 4Click Import Project.
  5. 5Select the project to import from. All its files will be copied into the destination project's source folder.
CLI — Command Prompt 
C:\> codespace import <destinationProjectName> <importingProjectName>

Run a Project

Run any project using the command you configured at setup — no need to navigate to its folder manually.

Run via GUI GUI CLI

  1. 1Open Codespace.
  2. 2Click on the project you want to run.
  3. 3The project dashboard will open.
  4. 4Click Settings.
  5. 5Click Run.
CLI — Command Prompt 
C:\> codespace run <projectName>

Open a Project in Your IDE

Open any project directly in whichever IDE you have configured — VS Code, JetBrains IDEs, Sublime Text, or any other editor. Works with any language: Python, Java, JavaScript, C++, C, HTML, and more.

Open via GUI GUI

  1. 1Open Codespace.
  2. 2Click on the project you want to open.
  3. 3The project dashboard will open.
  4. 4Click Open in IDE.
  5. 5Select your IDE from the list of configured editors.
⚠️ No CLI command available for this feature. Use the GUI to open projects in your IDE.
Projects

Export a Project to ZIP

Package any project's src directory as a .zip file — useful for sharing, backing up, or moving a project. The ZIP is saved in the same folder as the project's root directory.

Export via GUI GUI CLI

  1. 1Open Codespace.
  2. 2Click on the project you want to export.
  3. 3The project dashboard will open.
  4. 4Click Settings.
  5. 5Click Export to ZIP. The ZIP is saved alongside the project's root directory.
CLI — Command Prompt 
C:\> codespace exportZIP <projectName> <destinationFolder>

# Example
C:\> codespace exportZIP MyPythonApp "C:\Backups"

Automations

Automate Tasks in Windows

Automations let you automate tasks on your computer by attaching a Command Prompt command to a trigger action — before or after a project runs. You can automate tasks with Python, bash, Node.js, PowerShell, or any language that runs natively on Windows. This is the most powerful way to reduce repetitive work when handling multiple projects simultaneously.

Create an Automation GUI CLI

  1. 1Open Codespace.
  2. 2Select the Automations tab.
  3. 3Click Add.
  4. 4Enter an Automation name.
  5. 5Enter the name of the project or automation this should run before or after. If it triggers after a project runs, enter the project name.
  6. 6Check After Action to true if this automation runs after the action, or uncheck it to run before.
  7. 7Enter the Command Prompt command to run, including a cd to navigate to the automation directory.
  8. 8Click OK.
CLI — Command Prompt 
C:\> codespace automate <automationName> <command> <afterAction: true|false> <actionName>

# Example — run a Python cleanup script after "MyApp" runs
C:\> codespace automate Cleanup "cd C:\Scripts && python cleanup.py" true MyApp

Run on Startup / Shutdown

To automate tasks every time Codespace opens or closes, assign an automation in Settings:

  1. 1Open Codespace.
  2. 2Go to the Settings tab.
  3. 3Choose your automation under Run on Open or Run on Close.
Automations

Run an Automation Manually

Trigger any automation on demand directly from the Automations tab.

Run via GUI GUI CLI

  1. 1Open Codespace.
  2. 2Select the Automations tab.
  3. 3Click on the automation you want to run.
  4. 4The automation details will appear.
  5. 5Click Run.
CLI — Command Prompt 
C:\> codespace runAutomation <automationName>

IDEs

Configure an IDE

Register any installed editor so Codespace can open projects directly in it. Supports VS Code, JetBrains IDEs (IntelliJ, PyCharm, WebStorm), Sublime Text, and any other editor with a command-line interface.

Add an IDE GUI

  1. 1Open Codespace.
  2. 2Select the IDEs tab.
  3. 3Click Configure IDE.
  4. 4Choose the .exe file path of the IDE you want to add.
  5. 5Enter a name for the IDE.
  6. 6If the IDE uses a non-standard open command (other than <exe> <projectFolder>), enter the full command — replacing the exe path with <Path> and the project folder with <Project> (case-sensitive).
  7. 7Click OK.
⚠️ No CLI command available for configuring IDEs. Use the GUI only.
CLI Reference

CLI Commands for Managing Multiple Coding Projects

The Codespace CLI lets you manage multiple coding projects in Python, Java, JavaScript, C++, C, HTML, or any language directly from your terminal. Every command outputs success. on successful execution.

Command Description Arguments
codespace create Create and register a new project <name> <rootDir> <srcDir> <runCmd>
codespace delete Remove a project from Codespace <name>
codespace run Run a project using its configured command <name>
codespace import Copy files from one project's src into another <destination> <source>
codespace exportZIP Export a project's src folder as a .zip <name> <destinationFolder>
codespace get Retrieve a project's metadata JSON (ready to import) <name>
codespace automate Create an automation tied to a project or action <name> <cmd> <afterAction> <actionName>
codespace runAutomation Manually trigger an automation <automationName>
codespace deleteAutomation Delete an automation <automationName>
💡 Retrieve a project's metadata with codespace get <name> — it outputs a JSON object ready to import into another Codespace instance.
